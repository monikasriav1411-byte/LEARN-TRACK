# Learn Track

## Overview
Learn Track is a web-based student quiz and learning management platform designed to simplify the interaction between administrators, teachers, and students. The system provides role-based access and helps manage quizzes, track performance, and organize student learning activities efficiently.

## Features

### Admin Module
- Manage teachers
- Manage students
- Generate login credentials
- Department-wise student management

### Teacher Module
- Create and manage quizzes
- Assign quizzes to students from question bank
- Automatic question selection pattern.
- View student performance
- Track quiz completion status

### Student Module
- Secure login
- View available quizzes
- Attempt quizzes online
- Check quiz completion status
- Monitor performance and progress
